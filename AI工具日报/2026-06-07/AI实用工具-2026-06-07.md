## Kimi Code — 月之暗面出手，TypeScript 重写的终端编程 Agent

  Kimi Code 是 Moonshot AI 本周（6 月 6 日）正式发布的终端 AI 编程 Agent，它是原 kimi-cli（Python，6400+ stars）的 TypeScript 重写版本。Moonshot 把它定位为 Claude Code 的开源竞品——MIT 协议、npm 安装、支持 Kimi 模型也兼容其他 provider。

  核心思路是「feedback-driven execution」：Agent 读代码 → 规划步骤 → 修改文件 → 跑测试验证 → 根据结果调整，形成闭环。只读操作默认自动执行，文件编辑和 Shell 命令需要确认，保持开发者对代码的控制。

  这次重写带来的几个值得关注的能力：内置 `coder`、`explore`、`plan` 三个子 Agent，可以并行派发到独立上下文中执行；MCP 服务器通过 `/mcp-config` 对话式配置，不需要手写 JSON；支持视频输入——把屏幕录制拖进聊天窗口让 Agent 理解你的操作流程；Lifecycle hooks 让你在 tool call 前后插入自定义逻辑；通过 ACP 协议接入 Zed 和 JetBrains IDE。

```bash
# 推荐安装方式（无需预装 Node.js）
curl -LsSf https://code.kimi.com/kimi-code/install.sh | bash

# 或通过 npm（需要 Node.js ≥ 24.15.0）
npm install -g @moonshot-ai/kimi-code

# 启动
cd your-project
kimi
# 首次运行输入 /login 完成认证
```

## agent-qa — 用自然语言写测试，Agent 自己学会不再犯同样的错

  agent-qa 是一个带记忆系统的 AI 测试编排工具。你在 YAML 里用自然语言描述测试步骤，它驱动 Playwright（Web）或 Appium（移动端）去执行。关键在于它的 Memory 机制：每次测试跑完，agent-qa 会记录产品行为、测试观察和修复路径，下次再跑同样的测试时，这些记忆会被注入上下文，让 Agent 自动避开之前踩过的坑。

  传统 E2E 测试最大的痛点不是写，而是维护——UI 改一行，测试崩一片。agent-qa 的思路是用 AI 来吸收这个维护成本：当元素定位失败时，Agent 根据记忆和页面上下文自行修复选择器，而不是直接报错让你手动改。

  另外几个实用功能：支持在 Docker 沙箱里跑 Node/Bun/Python/Bash hooks，用来准备测试环境或清理数据；提供 MCP 和 Skill 接口，Claude Code、Cursor 等编程 Agent 可以直接调用它来验证代码变更；所有测试、配置、记忆都以版本控制友好的文件形式存储，diff 可审查。

```bash
# 安装为项目开发依赖
npm install -D agent-qa

# 初始化并安装浏览器驱动
npx agent-qa init
npx agent-qa install-browsers --chromium

# 启动 Dashboard
npx agent-qa dashboard --open

# 命令行运行测试
npx agent-qa run tests/homepage.test.yml
```

## CodeMate AI（C0）— 先索引代码库，再回答问题的 VS Code Agent

🏪 Visual Studio Marketplace · ⬇ 79,971 次安装 · (35) · 免费

  C0（CodeMate AI）是一个 VS Code 扩展形式的 AI 编程 Agent。它和其他 AI 编程助手的核心差异在于：「先建知识库，再做问答」——安装后会先对整个项目做索引，构建代码结构图谱，之后的每一次对话、调试、代码生成都基于对项目的完整理解。

  这种知识库优先的做法有几个实际好处：跨文件重构时 Agent 清楚每个符号的引用关系，不会漏改调用方；调试时可以追溯到调用链的源头而不是只看报错那一行；代码审查时能按模块依赖关系而非文件字母序来判断改动影响面。

  支持的语言超过 100 种，从 Python/C++/Java 到 Ruby/Dart/SQL，覆盖主流前后端技术栈。支持的框架包括 React、Next.js、Vue、Svelte、Angular、Node.js、FastAPI、Django、Spring 等。

  安装后在 VS Code 扩展面板搜索「CodeMate AI」即可，无需额外注册或 API Key 配置（免费额度包含在内）。
