## FluentCleaner — CCleaner 的精神续作，去掉了所有不该有的东西

### 它解决什么问题

  Windows 的 C 盘空间焦虑几乎是每个用户的必修课。微软自己都说存储空间不足会拖慢系统甚至阻止更新。你需要的工具很简单：扫描出哪些文件是可以安全删除的垃圾，让你勾选确认后一键清掉。CCleaner 曾经就是这样做的，但后来它变成了一个卖 VPN、弹升级广告、偷偷改注册表的不可信软件。

  FluentCleaner 做的事情极其简单：它读入社区维护了 15 年的 winapp2.ini 清理规则库（数千条应用缓存路径定义），逐条扫描你的磁盘，告诉你每条规则能清出多少空间，然后你选择清理哪些。没有注册表清理（那是伪优化），没有安全擦除（SSD 上那是玄学），没有后台服务，没有遥测上报。

### 核心原理

  FluentCleaner 的核心是两样东西：

  **winapp2.ini 解析器**：这是一个社区维护的纯文本数据库，每条记录定义了一个应用在 Windows 上的缓存路径。比如 Chrome 的缓存目录、微信的文件缓存、VS Code 的扩展缓存、npm 的全局缓存目录。FluentCleaner 把它解析成可勾选的清理条目，你看到的每一条都是精确的目标路径，不会出现"扫描出 20GB 垃圾但不敢删"的情况。

  **WinUI 3 原生界面**：不像大多数清理工具用 Electron 套壳或古董 WinForms，FluentCleaner 用的是微软最新的 WinUI 3 框架。启动快、渲染流畅、深色/浅色主题自适应，和 Windows 11 的视觉语言完全一致。

  **设计哲学上**：它故意不做注册表清理（对性能零影响，但误删会炸系统）、不做安全擦除（SSD 的 TRIM 和磨损均衡让多次覆写毫无意义）、不做一键优化（那是改几个注册表值自欺欺人）。它只做一件事——清掉确凿的垃圾文件。

  从 GitHub Releases 下载最新版，解压即用，绿色免安装：

```powershell
# 1. 下载最新版本
# https://github.com/builtbybel/FluentCleaner/releases

# 2. 解压到任意目录（如 D:\Tools\FluentCleaner）

# 3. 运行 FCleaner.exe，首次建议右键 → 以管理员身份运行
```

  可选步骤——更新清理规则库：

```powershell
# 下载官方 winapp2.ini 规则库（社区维护，覆盖数千应用）
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/MoscaDotTo/Winapp2/main/winapp2.ini" -OutFile "$env:AppData\FluentCleaner\winapp2.ini"
```

  然后在设置中：Settings → Database → Custom → 指向刚才下载的 winapp2.ini 文件。

### 使用指南

#### 场景一：首次深度大扫除

  你刚装上 FluentCleaner，C 盘红了很久。右键以管理员身份运行 FCleaner.exe，默认会加载内置清理规则。点击 Scan 按钮，等待扫描完成。你会看到每条规则后面标注了可释放的空间大小。逐条浏览，取消勾选你不确定的内容（一般来说都可以安全删除），然后点击 Clean 执行清理。

  我第一次用的时候扫出了 8.7GB 的垃圾，包括半年没清理的微信文件缓存（3.2GB）、Chrome 的 Code Cache（1.5GB）、Windows 更新残留（2.1GB），以及各种临时文件。

#### 场景二：定时自动清理

  设置好了清理选项后，不需要每次都手动打开。利用 `/AUTO` 参数配合 Windows 计划任务，让它每周自动静默清理一次：

  1. 打开「任务计划程序」（搜索 Task Scheduler）
  2. 创建基本任务 → 名称填"FluentCleaner 每周清理"
  3. 触发器 → 每周 → 选一个你不怎么用电脑的时间（如周六凌晨 3:00）
  4. 操作 → 启动程序 → 程序填 `FCleaner.exe` 的完整路径
  5. 参数填 `/AUTO`
  6. 勾选「使用最高权限运行」

  之后每次运行都会追加日志到 `%AppData%\FluentCleaner\auto.log`，记录了删了哪些文件、释放了多少空间。

#### 场景三：下班前清理+关机

  这个组合技特别适合下班时用——清理完所有垃圾后自动关机：

```powershell
FCleaner.exe /AUTO /SHUTDOWN
```

  放在桌面或任务栏，下班走人前点一下，电脑清理完自动关机。清理了什么、清了多少，下次开机去看 auto.log 就知道。

### 进阶技巧

  **自定义规则库**：FluentCleaner 支持多种 winapp2.ini 变体。除了官方社区版本，BleachBit、BBleachBit 等工具也有自己的规则变体。你甚至可以自己写规则文件——格式就是标准的 winapp2.ini，每一条是 `[应用名]` + `Section=路径|文件匹配|递归|删除参数`。有特殊清理需求的应用直接用这个自定义。

  **配合 FluentTweaker**：同一位作者还做了一个叫 FluentTweaker 的 Windows 调校工具，同样的 WinUI 3、同样的开源透明。FluentCleaner 负责清理，FluentTweaker 负责调校——比如关闭不必要的动画效果、优化资源管理器行为、管理右键菜单。两个一起用就是一套完整的 Windows 维护方案。

### 常见踩坑

  **Q: 扫描出来的某些条目是 0 字节，正常吗？**
  A: 正常。winapp2.ini 包含数千条规则，不是每条在你的机器上都有数据。0 字节的条目说明这个应用你没装或缓存已清空，不影响扫描结果。

  **Q: 清理后某些应用变慢了？**
  A: 正常现象。缓存的作用就是加速下次访问，清掉之后第一次打开会重新生成。日常清理不需要太频繁，每月一次就够了，保留最近的缓存不影响日常使用体验。

  **Q: 内置规则感觉不够多？**
  A: 下载社区完整版 winapp2.ini（见上方安装配置），内置规则只是精选常用的几十条，完整版有几千条。
