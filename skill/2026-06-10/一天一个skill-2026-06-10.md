# SkillAlchemy — 一念落地，万象成形

  2026 年大家都在装 skill、用 skill，但有没有想过——skill 本身是怎么来的？把一个人的思维方式、一套方法论、一段领域经验变成一个 Agent 可以直接加载的 SKILL.md，这件事之前全靠人工手写。SkillAlchemy 把这个过程自动化了：你告诉它一个名字或一个主题，它派出多个研究 Agent 并行深挖，蒸馏出一个结构完整、可直接安装的 skill。

---

### 它解决什么问题

  写 skill 的人都知道最难的不是格式，是**内容萃取**。你想把"费曼学习法"做成一个 skill——翻书、整理、写 prompt、测试效果、反复修改……一个高质量的 skill 可能需要好几个小时。更不用说你想蒸馏一个人物（比如张雪峰的高考志愿填报逻辑），那需要大量资料搜集和结构化。

  SkillAlchemy 把这个过程变成一句话的事。它不是一个"帮你写 prompt"的工具——它是一套**自动化的知识蒸馏流水线**，专门把 raw source（人、方法论、书籍、仓库）转化为 Agent 可执行的结构化 skill。

### 核心原理

  SkillAlchemy 的核心是一条**多 Agent 并行研究 → 编译验证**的蒸馏管线：

  ```
  用户输入（名字/主题）
       ↓
  Lens 分析（判断蒸馏类型：人物/方法论/技能融合）
       ↓
  Research Swarm（3-8 个 Agent 并行深挖）
       ├─ 决策时刻
       ├─ 失败处理模式
       ├─ 价值观冲突
       └─ 表达风格
       ↓
  Gate 1 质量审查（内容覆盖度检查）
       ↓
  Exemplar（在线检索同类 skill 作为参考）
       ↓
  Compile（编译 SKILL.md + 自评 + 验证）
       ↓
  产出：可安装的 skill 目录
  ```

  三种蒸馏模式：
  - **蒸馏一个人**：从公开资料中提取决策逻辑、表达风格、价值观取向，编译成一个 Persona Skill，可以随时"对话"
  - **蒸馏方法论**：一本书、一套方法论、一个开源仓库——蒸馏成可执行 SOP：前置条件 → 执行步骤 → 分支判断 → 失败处理，每一步都有可验证的证据
  - **融合已有 Skill**：把两个 skill 组合成一个新能力——比如「第一性原理 × 费曼学习法」→ 一个拆到底、讲到懂的深度理解引擎

  只有一个参数 `depth`，控制 Agent 数量和验证严格度：

  | 维度 | quick | standard | deep |
  |------|-------|----------|------|
  | Agent 数 | ≤3 | 4-5 | 6-8 |
  | 预计耗时 | ~5-8 min | ~15-20 min | ~25-35 min |
  | 内容验证 | 跳过 | 建议 | 强制 + 双审核 |
  | 标注 | `draft` | `standard` | `validated` |

### 安装配置

  ```bash
  # 命令行安装（推荐）
  npx skills add agentsope/SkillAlchemy

  # 或者直接让 Claude Code 帮你装：
  # "请帮我从 https://github.com/agentsope/SkillAlchemy 拉取这个 skill 到本地，安装到 Claude Code 的 skills 里"
  ```

  安装完成后重启 Claude Code，SkillAlchemy 会根据对话内容自动激活。你也可以显式调用。

  仓库的 `skills/` 目录下还包含大量已经蒸馏好的 skill，不需要从头蒸馏也能直接用：

  ```bash
  # 安装一个已蒸馏好的 skill（如 Lens）
  npx skills add agentsope/SkillAlchemy/skills/lens
  ```

### 使用指南

  #### 场景一：蒸馏一个领域专家的人格 skill

  你想做一个"张雪峰"风格的 skill，帮普通家庭分析高考志愿填报逻辑。不需要自己去搜集他的所有直播切片和文章——一句话交给 SkillAlchemy：

  > 蒸馏「张雪峰」→ persona skill，用 standard 深度

  系统派出 5 个研究 Agent 并行工作：一个分析他的决策框架（学校优先还是专业优先）、一个提取他的典型话术和表达风格、一个整理他的价值观判断（实用主义 vs 理想主义）、一个收集他的失败案例和反思、一个做交叉验证。Gate 1 审查通过后编译 SKILL.md，自评质量达标后产出。

  全程约 15-20 分钟，你只需要在任务简报和 research plan 两个节点确认一下。

  #### 场景二：把一本书或一套方法论蒸馏成可执行 skill

  读完《思考，快与慢》，想把卡尼曼的双系统理论变成一个日常决策辅助 skill：

  > 蒸馏「思考快与慢」中的双系统决策框架 → methodology skill，用 deep 深度

  deep 模式下 6-8 个 Agent 并行，强制双审核验证。产出的 skill 不是"这本书讲了什么"的摘要，而是可执行的 SOP：「当你面临 X 类决策时，先检查是否触发了系统 1 的 Y 类启发式偏误，如果是，激活系统 2 的 Z 类分析框架」。

  每个步骤都有书中原文锚点做证据支撑。

  #### 场景三：融合两个已有 skill 创造新能力

  你有一个「第一性原理分析」skill 和一个「费曼学习法」skill，想把它们组合成一个"拆解 + 讲懂"的深度学习引擎：

  > 融合「第一性原理」和「费曼学习法」两个 skill → 新的深度理解 skill

  SkillAlchemy 会分析两个 skill 的结构，找到结合点（第一性原理负责拆到底，费曼负责讲到懂），生成一个新的复合 skill。不是简单拼接——它会识别两个方法论之间的衔接逻辑，定义新的前置条件、执行顺序和输出标准。

  #### 场景四：从开源仓库蒸馏最佳实践

  你在研究一个大型开源项目的架构模式，想把它的设计原则蒸馏成团队可复用的 skill：

  > 蒸馏「React 源码中的调度架构模式」→ methodology skill

  SkillAlchemy 的研究 Agent 会分析源码结构、commit 历史中的设计决策、核心贡献者的讨论，提取出可迁移的架构原则。

### 进阶技巧

  **深度参数的选择策略**：quick 模式适合快速验证一个蒸馏想法是否可行（先花 5 分钟跑一次，看产出质量决定要不要 deep）。standard 是日常推荐——性价比最高，Agent 数量刚好覆盖主要维度不遗漏。deep 适合你要公开发布、其他人会依赖的 skill，双审核保证可靠性。

  **搭配仓库已蒸馏 skill 使用**：SkillAlchemy 仓库自带了一批蒸馏好的 skill（如 Lens、各种框架适配器），在 `skills/` 目录下。先 `npx skills add` 装几个现成的，理解它的产出标准，再用它蒸馏自己的 skill——有参照物之后，你对产出质量的判断会更准。

### 常见踩坑

  **不适合资料稀缺的人物复刻**：别指望三条新闻就召唤出一个完整的人格。SkillAlchemy 的研究 Agent 依赖公开资料的丰富度。如果一个人几乎没有公开言论、访谈、文章，产出的 persona skill 会很空洞。做之前先搜一下，确认有足够的公开素材。

  **deep 模式需要耐心**：25-35 分钟的等待时间不算短，而且中间有两个确认点需要你参与。如果你离开电脑去倒杯水，回来发现卡在 Gate 1 等你确认，会浪费不少时间。建议用 standard 模式先跑，看效果再决定是否加深度。

  **不适合版权内容改写复用**：换个发型不等于换了个人，洗稿也不等于原创。SkillAlchemy 蒸馏的是公开可用的方法论和思维框架，不是让他人版权的文字换种说法。

### 同类技能对比

| 技能 | 安装方式 | 核心特点 | 适用场景 | 热度 |
|------|---------|---------|---------|------|
| **SkillAlchemy**（今日推荐）| `npx skills add` | 多 Agent 并行蒸馏、三种模式、已附带预蒸馏 skill 库 | 快速创建高质量 skill，知识萃取 | ⭐ 66 |
| skill-creator (Anthropic 官方) | `/plugin install` | 手把手引导写 skill、模板生成、验证 | 从零手写 skill | 内置 |
| Skill Forge (Adit-Jain-srm) | `npx skills add` | 18 个预置 skill + 自优化循环 | 快速获取预置 skill 套装 | ⭐ 1 |

  SkillAlchemy 的独特定位是**知识蒸馏的自动化**。官方 skill-creator 帮你规范地手写 skill，SkillAlchemy 帮你从 raw source 自动萃取内容。两者互补：用 SkillAlchemy 快速出初稿，用 skill-creator 精修打磨。Skill Forge 提供预置 skill 但蒸馏能力不在一个量级。

## 今日小结

  SkillAlchemy 适合每个正在建设自己的 skill 库的开发者。如果你已经感受到了"好 skill 太少、写 skill 太慢"的瓶颈，这个工具能把创作速度提升一个数量级——从几小时手写到几分钟自动蒸馏。它最核心的价值不是"又一个 skill"，而是"让你的 skill 生产能力工业化"：有了它，任何你遇到的好方法、好经验、好思维框架，都能在 20 分钟内变成一个可复用、可分享的 skill。花 2 分钟装上，今天就可以开始蒸馏你的第一个 skill。
