## 2. JetBrains Mellum2 开源：12B MoE 代码专用模型，Apache 2.0

**评分: 8.6/10 | 可操作性: 极高（HuggingFace 直接下载，Apache 2.0 可商用）| 热度: JetBrains 官方博客 · HN 首页**

**是什么：**
JetBrains 于 6 月 1 日正式开源 Mellum2，一款专为代码场景设计的 MoE（混合专家）模型。关键参数：12B 总参数 / 仅 2.5B 激活参数每 token，专注自然语言和代码数据训练，不支持多模态。Apache 2.0 许可证可商用。设计哲学是"focal model"——放弃通用多模态能力换取极致的代码理解和低延迟推理。

**怎么用：**
```bash
# 从 HuggingFace 下载
pip install transformers torch
huggingface-cli download JetBrains/Mellum2

# Python 推理示例
from transformers import AutoModelForCausalLM, AutoTokenizer
model = AutoModelForCausalLM.from_pretrained("JetBrains/Mellum2")
tokenizer = AutoTokenizer.from_pretrained("JetBrains/Mellum2")
inputs = tokenizer("def fibonacci(n):", return_tensors="pt")
outputs = model.generate(**inputs, max_new_tokens=100)
print(tokenizer.decode(outputs[0]))
```

**为什么重要：**
在模型日益臃肿的 2026 年，JetBrains 选择了"做减法"的路线——12B 总参 / 2.5B 激活参数意味着可在消费级 GPU 上轻松运行。Apache 2.0 许可证使其成为企业友好的本地部署选项。这是一个明确的信号：AI 工具链厂商开始自研模型优化自家产品体验，而非单纯依赖第三方 API。

**来源链接：**

**怎么用：**
```bash

# 在 .claude/mcp.json 中添加：
{
  "mcpServers": {

      "command": "npx",
      "args": ["@saltsecurity/salt-code-mcp"]
    }
  }
}

# 支持 GitHub / GitLab / Bitbucket CI 集成
# 详细配置见官方文档
```

**为什么重要：**
2026 年 AI 编码助手已覆盖近半数代码产出，但安全治理严重滞后。Salt Code 的出现填补了"AI 写代码，谁来保证安全"的空白。它不是另一个 SAST 工具，而是直接嵌入 AI 助手的实时安全策略引擎。对于使用 Claude Code 或 Copilot 的团队，这是目前最直接的安全兜底方案。

## 6. GitHub Copilot 正式切换 Token 计费：15 倍请求乘数引发开发者强烈反弹

**评分: 7.8/10 | 可操作性: 中（被动承受，需调整使用策略）| 热度: Instagram/X/Reddit 刷屏 · 开发者社区爆发式讨论**

**是什么：**
GitHub 于 6 月 1 日正式将 Copilot 从固定月费切换到 Token 计费模式。关键变化：(1) 高级请求消耗 token 速度远超预期；(2) 对特定操作施加 15 倍请求乘数；(3) Business 和 Enterprise 用户获 3 个月过渡额度（Business +$30/月，Enterprise +$70/月）；(4) 组织级别支持 credit pool 按成本中心分配。开发者普遍担忧账单失控。

**怎么用：**
```bash
# 查看当前 Copilot 使用量和计费
# GitHub Settings > Copilot > Usage

# 建议立即配置使用上限
# Organization Settings > Copilot > Policies > Usage limits
# 设置月度 token cap 防止账单暴增

# 替代方案评估
# - 继续使用 Copilot 但限制 agent 模式
# - 切换到 Claude Code（Anthropic API 直连计费透明）
# - Cursor 仍保持月费模式
```

**为什么重要：**
这是 AI 编码助手商业模式的重大转折点。固定月费时代的结束意味着开发者需要像管理云资源一样管理 AI token 消耗。对于重度 agent 使用场景、长上下文编程的开发者而言，成本可能从 $10-39/月暴增到数百美元。社区已经开始寻找替代方案。
