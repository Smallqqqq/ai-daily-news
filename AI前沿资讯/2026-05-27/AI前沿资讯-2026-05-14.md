## 1. Anthropic "Code with Claude 2026" — Agent 三大新功能

**评分: 9.8/10 | 影响力: 直接改变 AI 编码工作流**

5月6日 Anthropic Dev Day 发布三项 Claude Managed Agents 新能力:

- **Multi-agent Orchestration** — 多 Agent 编排，创建 Agent 舰队拆分执行复杂任务，每子 Agent 可配置不同角色和工具
- **Outcomes** — 用评分 Rubric 定义"成功标准"，独立评分 Agent 评估输出质量，不达标自动打回重做
- **Dreaming (autoDream)** — Agent 跨会话记忆整合，检查之前遗漏并自我提升，合并重复记忆、剔除过期信息

同步发布:
- Claude Code Desktop 重设计 — session 侧边栏、拖拽工作区、集成终端和文件编辑器、macOS SSH 支持
- Claude Code Routines — 定时/Webhook 触发自动化工作流
- Claude for Legal + Claude for Finance — 垂直行业 Agent 套件
