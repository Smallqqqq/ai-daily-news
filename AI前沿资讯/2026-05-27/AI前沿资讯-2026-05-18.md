## 3. OpenHuman — 开源个人 AI 超级智能体

**评分: 8.8/10 | 可操作性: 高 | 热度: GitHub #1 (+3,941 stars/day) · 总 18.3K**

**是什么：**
OpenHuman 是一个开源的桌面个人 AI 助手，由 tinyhumansai 用 Rust + TypeScript + Tauri 构建。核心特色是三件套：Memory Tree（≤3K token 的 Markdown 分块记忆）、TokenJuice（智能 token 压缩，声称降 80% 成本）、118+ OAuth 集成（Gmail、Notion、GitHub、Slack、Stripe 等）。内置本地优先的 Obsidian 兼容知识库，每 20 分钟自动从连接的服务拉数据。

**怎么用：**
```bash
# macOS/Linux
curl -fsSL https://raw.githubusercontent.com/tinyhumansai/openhuman/main/scripts/install.sh | bash

# Windows
irm https://raw.githubusercontent.com/tinyhumansai/openhuman/main/scripts/install.ps1 | iex

# 支持 Ollama 本地模型或云端 API
```

**为什么重要：**
单日近 4,000 stars，超过第二名（学术研究技能包 1,439）和第三名（隐身浏览器 1,420）之和。个人 AI 主权 + 本地优先 + 全服务打通，正好踩中 2026 年隐私和个人 AI 需求爆发的节点。

**讨论现场：**
- GitHub Issues 中用户最关注：中文支持完善度、与 Claude Code 的协同使用场景
- HN 评论："这可能是第一个真正可用的本地个人 AI 操作系统"

**来源：** https://github.com/tinyhumansai/openhuman

## 5. CloakBrowser — 隐身 Chromium，过所有 Bot 检测

**评分: 8.3/10 | 可操作性: 高 | 热度: GitHub #3 (+1,420 stars/day) · 总 15.6K**

**是什么：**
CloakHQ 开发的隐身 Chromium 浏览器，在源码层面修补指纹检测点，可作为 Playwright 的即插即用替代。声称通过所有主流 Bot 检测测试（Cloudflare、Datadome、Akamai 等）。

**怎么用：**
```bash
git clone https://github.com/CloakHQ/CloakBrowser
cd CloakBrowser
pip install -e .
# 作为 Playwright 替代
from cloakbrowser import Browser
browser = Browser()
page = browser.new_page()
page.goto("https://example.com")
```

**为什么重要：**
AI Agent 做 Web 自动化最头疼的就是被检测拦截。CloakBrowser 在 Chromium 源码层打补丁，绕过所有主流 Bot 检测，对依赖网页抓取的 AI Agent 工作流（数据采集、竞品监控、自动化测试）是基础设施级工具。+1,420 stars/day 说明这个痛点非常普遍。

**讨论现场：**
- 社区讨论：与 playwright-stealth 的对比——CloakBrowser 在源码层修而不是 JS 层绕过，更彻底
- 有用户提醒：用于合法场景（测试/爬取公开数据），滥用可能触犯服务条款

**来源：** https://github.com/CloakHQ/CloakBrowser
